Understanding the Box Model

Every element in an HTML document is treated as a rectangular box. The CSS Box Model describes how the dimensions of these boxes are calculated, including their content, padding, borders, and margins. This is fundamental to all of web layout.

Key Components

  • Content: The actual text, images, or other content of the element.
  • Padding: The space between the content and the border. Padding is an inner space that adds to the box's size.
  • Border: The line that surrounds the padding and content.
  • Margin: The space outside the border. Margin is an outer space used to create separation between elements.
